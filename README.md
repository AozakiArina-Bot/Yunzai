# 说明

此仓库为个人fork版本自用

## 安装

个人设置 -> 访问令牌 -> 新建访问令牌
```
git clone --depth=1 https://github.com/AozakiArina-Bot/Yunzai.git
```

换源
```
git remote set-url origin https://github.com/AozakiArina-Bot/Yunzai.git
git fetch origin master
git reset --hard origin/master
```
